This BMI calculator executable program is a simple application of functions studied in C language.
It includes use of printf, scanf, else if, etc. functions and also show nested structure.

This program is precisely giving results according to the latest information provided by 
health ministry about BMI.

It gives a standard result which is irrespective of gender and age.

The objective to create such basic application is to create interest in learning C- language by the method of 
learning by doing.
